# p5-mojolicious-image-bbs
Mojoliciousで作る画像掲示板

## 実行環境がある場合
### 画像掲示板をローカル環境で起動する
```
$ git clone https://github.com/nqounet/p5-mojolicious-image-bbs.git
$ cd p5-mojolicious-image-bbs
$ cpanm --installdeps .
$ morbo script/my_app
```

## 実行環境がない場合
起動できなかった場合は何かが足りないので実行環境を整える
